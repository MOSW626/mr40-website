# 인앱 브라우저 실기기 테스트

대상: iPhone Safari/카카오톡, Android Chrome/카카오톡/네이버 앱

- 단톡방 링크에 제목, 설명, 1200×630 대표 이미지가 표시된다.
- 320px 폭에서 상단 `☰ 메뉴`와 하단 4개 탭이 겹치지 않는다.
- `가+`를 누른 뒤 새 페이지로 이동해도 큰 글자 설정이 유지된다.
- 확대 제스처와 브라우저 글자 크기 변경이 동작한다.
- 사진관 첫 진입 시 최신 연도가 선택되고 스크롤할 때 사진이 추가된다.
- 사진을 연 뒤 휴대폰 뒤로가기를 누르면 사이트 이탈 없이 사진만 닫힌다.
- 주소록 접속 코드 입력 시 숫자 키패드가 표시된다.
- 전화 링크, 지도 링크, 외부 새 창 안내가 올바르게 동작한다.
- 설문 iframe 입력·제출이 동작한다. 실패 시 새 창 버튼으로 제출할 수 있다.
- 캘린더 파일이 내려받아지고 날짜가 2026년 11월 28일로 표시된다.
- 결제·지도·라이브 URL이 없을 때 버튼은 비활성 준비 중 상태다.
- 공유 버튼이 시스템 공유창을 열거나 URL을 클립보드에 복사한다.
